Creative analytics
To make it easier for you to navigate through our platform, we have divided the ‘Getting Started’ section into two parts:
  1. Evaluating Ads without Campaign information
  2. Creating Campaigns and evaluating Ads within the Campaign
Part 1: Evaluating Ads
  • - As soon as you login, you land on the “Creatives” page.
  • - Top right corner has 2 options: Create campaign and Upload creatives
  • - Click on “Upload creatives”
  • - The following information is required to fill in:
    • Creative name (mandatory field)
    • Creative type (mandatory field): Creative types can be either Video Ad or Display Ad
    • Select brand (mandatory field): You can select only those Brands that you have access to. The brand selected for the Ad evaluation ensures that the Brand Guidelines are applied for the analysis.
    • Drag and drop the file or browse files on your computer to select an Ad file to analyze
  • - Click on “Upload and analyze” so the file is uploaded and put into analysis
  • - Click on “Upload” if you wish to run the analysis later
  • - Click on “Cancel” if you uploaded a wrong file and entered wrong information
Please note: The creatives uploaded by you are visible to you only and not to any other team member who has access to Incivus platform. If you wish to share the creatives and its report with other team members, we recommend creating a Campaign and uploading Ads within the Campaign for analysis. Campaigns can be shared with other team members who have access to the brands mapped to the creative uploaded for analysis
Part 2: Creating a campaign folder
  • - As soon as you login, you land on the “Creatives” page.
  • - Top right corner has 2 options: Create campaign and Upload creatives
  • - Click on “Create campaign”
  • - Provide the following mandatory information
    • Campaign name: Ensure there are no special characters in the name you provide.
    • Brand: ou can select the brand which you have access to. The Ads uploaded in that Campaign will be mapped to this selected Brand Guidelines.
  • - Click on “create” to create the campaign
  • - Click on “cancel” to cancel creating this new campaign
  • - Understand what makes an ad memorable
  • - Sharing campaign
    • As the author of the campaign, you can share the campaign with other users of your organization who have access to the Brand for which the Campaign folder is created.
    • As the author of the campaign, you can delete the campaign folder if it is empty. If it is not empty, you will not be able to delete the campaign folder.
A/B Testing

So, you want to discover what truly works for your audience since you need data as evidence before you make a final decision. One of the most common methods of gathering this data is through A/B testing.

A/B testing, at its most basic, is a way to compare two versions of something to figure out which performs better. In our case it's about comparing 2 versions or sub-edits of an Ad to see which one will capture your audience’s attention and deliver for your marketing objectives.

A/B testing feature permissions are required on the Incivus Platform so you can test 2 versions of an Ad. If you do not see “A/B Testing” as a menu item in the left-hand navigation, you do not have permission to use this feature. Please reach out to your system admin or write to Incivus at support@incivus.ai for further assistance.

Steps to using the A/B Testing feature:

  • - Pre-requisite to A/B Testing is to have the 2 versions of the Ad already analyzed on the platform.
  • - Click on “A/B Testing” on the left-hand navigation
  • - On the landing page of A/B Testing, you can filter the Ads displayed based on the type of the Ad. Select display ad or video ads from the filter to compare
  • - You can select the Ads displayed and then hit the “Compare” button on the top-right corner
  • - Click on “cancel” to cancel creating this new campaign
  • - The metrics are displayed in a tabular format for the selected Ads
  • - Click on “save” on the top-right corner to save this A/B test report
  • - All saved reports are listed in a tabular format in the A/B testing landing page. You can refer to them any time to make go/no-go decisions
Service Request

The users of the Incivus Platform such as the Brand Managers, Creative Directors or Growth Marketers can request for the following services from Incivus Analyst team:

  1. Create a new Ad
  2. Create Ad variations
To submit a new request, following pre-requisites are a must:
  • - Permission to service request feature
  • - If the request type is “Create new Ad”, the following information should be available with the user before they start creating the request
    • Ad brief
    • Brand for which the new Ad must be created
    • Brand Assets that should be used in the Ad
    • Social media platform channels for which this Ad must be created
    • Launch date
    • Colors that should be used as dominant colors in the Ad
    • Emotions that you would like the Ad to emote
  • - If the request type is “Create Ad variation”, the following information should be available apart from all the above information
    • An image or video Ad file for which a variation needs to be created
Steps to creating a service request:
  1. As the user with the permission to create a service request, you will find this feature under settings on the left-hand navigation.
  2. Click on “All” under “Service Requests” menu on left-hand navigation to load the page with the option to create a new request.
  3. Click on “+New request” to load a form that must be filled in for the new request to be generated.
  4. Provide a summary that is easy to understand.
  5. Request type should be selected as either “Create new Ad” or “Create Ad variation”.
  6. Select the brand and creative type as “video ad” or “display ad”. Note: Brands available here for you to select from are the ones you have access to.
  7. Ad brief can be provided in text or as attachment. Only upto 5 attachments are allowed for this field.
  8. You must select a channel for which the Ad must be created to ensure the resolution as well as other aspects of the file are compatible with the social media channel.
  9. The launch date provided here would help the Analyst team to service you on time.
  10. Brand assets that must be used for the Ad should be selected. If no brand assets are selected, then the assets from Brand Guidelines for the brand is used for creating the ad or its variation.
  11. For creating Ad variations, the Ad for which variations must be designed should be uploaded as part of Ad brief or Assets.
  12. Any additional details that might want to be added to the current request can be accommodated in the additional details field. The word limit here is 500 words.
  13. Click on the submit for confirmation.
  14. A service request number will be automatically generated.
  15. The service requests logged into the platform will be seen in all sections of the navigation panel.
  16. As soon as the service request sitting in the Task list is marked as complete, the status changes to “Completed”, In case there are additional details required on the ticket, it is put “on-hold”.
  17. Status of the service request can be seen on looking into the service id details or on the quick look table.
  18. Completed service requests will be seen in green color and in progress status will be observed in amber color.
  19. Any service request which is in “closed” state for more than 3 months goes into the “Archived” list. These closed service requests are available only for 24months (about 2 years) from the day it was raised. After this, the service request is removed from the system.
